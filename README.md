
# QuickCD - qcd

This command allows you to save directories under an alias,so you can cd to it quicker. Linux Only


## Usage/Examples

```
[~]$ qcd set /usr/bin ubin
[~]$ qcd ubin
[/usr/bin]$ pwd
/usr/bin
[/usr/bin]$
```
